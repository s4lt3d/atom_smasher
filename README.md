# Atom Smasher

> Interactive particle system visualization showing Coulomb forces between charged particles in Processing.

---

## Screenshots

![Atom Smasher Demo](https://github.com/s4lt3d/atom_smasher/blob/master/atom_smasher.gif?raw=true)

---

## Overview

A visual simulation of 150 particles that interact with each other based on their electrical charges. Each particle exerts Coulomb forces on all others, causing them to accelerate and move across the screen in complex patterns.

---

## Features

- **Particle Simulation** — 150 particles with individual charge properties
- **Physics-Based Interaction** — Coulomb force calculations between all particle pairs
- **Visual Feedback** — Real-time rendering of particle positions and movements
- **Frame Capture** — Save animation frames as images
- **Interactive Clearing** — Press mouse to clear the background

---

## Usage

**Requirements:** Processing 4.0 or later with JavaFX library

1. Open `atom_smasher.pde` in Processing
2. Click Run to start the simulation
3. Mouse click: clears the background
4. The animation will generate emergent patterns from the particle interactions
